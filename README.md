# Twinoid Goals

A simple, straight-forward package containing everything you need to handle Twinoid goals.

![](https://i.imgur.com/tHp0kBw.png)

## Installation
```
npm i twinoid-goals
```

## Usage

```ts
import { Games } from 'twinoid-goals';

console.log(Games);
```
